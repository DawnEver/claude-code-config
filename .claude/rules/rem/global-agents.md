# GLOBAL-AGENTS.md

Never modify `GLOBAL-AGENTS.md` in this repo. It is the single source for global guidance on
both hosts — `setup.js` symlinks it to `~/.claude/CLAUDE.md` (Claude Code) and
`~/.codex/AGENTS.md` (Codex's global instructions file). It applies globally across all projects.

**How to apply:** Only edit `AGENTS.md` for project-specific changes. Never open `GLOBAL-AGENTS.md`.
