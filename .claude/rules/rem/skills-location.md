# Skills Location

All custom skills and agents go in the repo at `skills/<name>/SKILL.md`, symlinked to `~/.claude/skills` by `setup.js`.

**Why:** Version-controlled, synced via OneDrive, survive cache clears.

**How to apply:** Write to `skills/<name>/SKILL.md` in the repo — never directly into `~/.claude/` or `~/.claude/plugins/cache/`.
